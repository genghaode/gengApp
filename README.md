# gengApp
## 目录结构
- app 
    * Entry.js 入口文件，redux配置
    * App.js 路由配置
    * api 网络请求
    * component 公共组件
    * container 包裹层组件
    * page 页面
    * redux 
    * store
    * util 工具库
## 使用
```
# 下载
$ git clone https://github.com/genghaode/gengApp.git
# 进入目录，安装依赖
$ cd gengApp && npm i
# 运行ios
$ react-native run-ios
```
